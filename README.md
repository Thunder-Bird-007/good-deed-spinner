# Good Deed Spinner

A beginner-friendly Python project built in GitHub Codespaces using Jupyter Notebook.

This app is inspired by random spin games, but redesigned in a meaningful and halal way. Instead of betting, it gives users useful tasks such as good deeds, study actions, and small productive challenges.

## Web App Version

This project also includes a Streamlit web app version.

Run it with
```bash
streamlit run app_v2.py

## Features
- Random 3-symbol spinner
- Match-based points system
- Random task generator
- Beginner-friendly Python logic
- Built step by step in notebook form
- Streamlit web interface
- persistent score using session state
- reset button

## Technologies Used
- Python
- Jupyter Notebook
- GitHub Codespaces

## Project Motivation
I wanted to learn Python with a fun random-based project while avoiding gambling-related themes. This project keeps the excitement of spinning while making the outcome beneficial.
